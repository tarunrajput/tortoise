## Installation

Clone Repo

```sh
$ git clone https://github.com/tarunrajput/tortoise.git
```

Navigate to Repo Directory

```sh
$ cd tortoise
```

Install Dependencies

```sh
$ yarn install
```

Run
```
$ yarn start
```
